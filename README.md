# Note App Vue Trainer

### Abstract
This is a trainer Project (Train Vue)

### Prerequisites
Before you continue, ensure you meet the following requirements:
- Installing a Code editor (ex: VS Code)
- Installing Nodejs

### Installation 
#### 1. Client folder
- Open Terminal
- Run `npm i`
- Run `npm run serve`

### Technologies
#### 1. Client folder
- Framework: `VueJS`
- UI lib: `ElementUI`
- Style: `SCSS`
