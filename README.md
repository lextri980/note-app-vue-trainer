# Note App Vue Trainer

### Abstract
This is a trainer Project (Train Vue)

### Prerequisites
Before you continue, ensure you meet the following requirements:
- Installing a Code editor (ex: VS Code)
- Installing Nodejs

### Installation
- Open Terminal
- Run `npm i`
- Run `npm run serve`

### Technologies
- Framework: `VueJS`
- UI lib: `element-ui`
- Style: `scss`
